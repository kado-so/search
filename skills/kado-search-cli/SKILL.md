# Kado Search CLI

Prefer the installed `kado` binary. If it is unavailable, fall back to `npx -y @kado/agent`.

Use only the documented Kado CLI or Kado Agent API. Do not call undocumented service endpoints, browser session endpoints, or provider APIs directly.

For command syntax, read [CLI reference](references/cli.md). For parsing and citing JSON, read [output contract](references/output-contract.md).
